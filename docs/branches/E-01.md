# E-01 — производная Enterprise Action Model

## Статус и границы

- **VERIFIED FACT:** Foundation checkpoint — `Qwen/Qwen3-Coder-Next-Base`; model card фиксирует 80B total, 3B activated, 48 слоёв, hidden size 2048, 512 routed experts, Top-10 и один shared expert, native context 262 144.
- **VERIFIED FACT:** E-01 является proprietary post-trained derivative, а не моделью, pretrained с нуля; происхождение и upstream notices сохраняются во внутренней model card и поставке.
- **VERIFIED FACT:** Любой model compute разрешён только в RunPod-кластере NVIDIA H200 HGX с NVLink/NVSwitch внутри узла. Локальный control plane не хранит веса и не выполняет training/inference.
- **EXPERIMENT REQUIRED:** Перед multi-node run allocation обязан предоставить attestation InfiniBand/high-bandwidth fabric и пройти NCCL acceptance.
- **RISK:** Внешняя документация не подтверждает fabric, цену, quota или sustained throughput конкретного RunPod allocation; без attestation и acceptance run запрещён.
- **VERIFIED FACT:** Эта ветвь не санкционирует deployment или training run; её artifact — спецификации, schemas, manifests, eval harness design и runbooks.

## Исследовательский вопрос

- **ENGINEERING HYPOTHESIS:** Узко обученный E-01 с CodeWorld, policy-gated runtime и executable verification превысит general-purpose frontier APIs по Verified Enterprise Task Completion Rate на фиксированном наборе enterprise coding workflows при меньшей стоимости разрешённой задачи.
- **EXPERIMENT REQUIRED:** Сравнить untouched Base, минимальный SFT checkpoint и полный E-01 candidate на одном frozen private holdout; публиковать доверительные интервалы, latency и стоимость, а не один aggregate score.

## Work packages

1. **VERIFIED FACT:** E01-WP0 замораживает baseline, LÆTEX-Bench protocol и contamination registry до post-training.
2. **ENGINEERING HYPOTHESIS:** E01-WP1 строит CPT corpus manifests и tokenizer-preserving packing без изменения vocabulary.
3. **ENGINEERING HYPOTHESIS:** E01-WP2 создаёт verified SFT trajectories с tool grammar, evidence contract и recovery paths.
4. **ENGINEERING HYPOTHESIS:** E01-WP3 обучает request-level Top-2 adapter router и shared Evidence/Policy adapter отдельно от internal token-level MoE.
5. **ENGINEERING HYPOTHESIS:** E01-WP4 реализует CodeWorld V0, Organizational World Model V0 и policy-gated action runtime.
6. **EXPERIMENT REQUIRED:** E01-WP5 допускает preference optimization и executable RL только после прохождения SFT safety/control baselines.
7. **VERIFIED FACT:** E01-WP6 выпускает checkpoint candidate, adapters, schemas, eval report, model card, SBOM, licenses и reproducibility manifest.

## Compute gate

- **ENGINEERING HYPOTHESIS:** Минимальная исследовательская конфигурация для ограниченных adapter/SFT экспериментов — один 8×H200 HGX node; рекомендуемая конфигурация для selective/full-parameter экспериментов определяется memory model и профилированием на 8 GPU, но не обещается заранее.
- **EXPERIMENT REQUIRED:** До первого оплачиваемого run выполнить dry-run конфигурации без весов, проверить container digest, topology, NCCL/IB, encrypted object storage, checkpoint resume и telemetry.
- **RISK:** Полный BF16 training state 80B невозможно экономически обосновать без измеренного uplift над adapters; full-tune запрещён до доказательства, что adapter ceiling блокирует release metric.
- **VERIFIED FACT:** Checkpoints пишутся в tenant-isolated object storage, versioned registry и immutable lineage ledger; локальная выгрузка весов запрещена.

## Exit criteria

- **EXPERIMENT REQUIRED:** Candidate увеличивает VETCR относительно untouched foundation не менее чем на заранее зафиксированный minimum detectable effect без ухудшения security critical-failure rate.
- **EXPERIMENT REQUIRED:** Pass-to-pass regression на frozen coding suite не хуже frozen non-inferiority margin `<=2 pp`.
- **EXPERIMENT REQUIRED:** 100% write actions имеют policy decision, scoped credential, evidence bundle и replayable audit entry.
- **RISK:** При отсутствии statistically defensible uplift, устойчивого recovery после tool failures или приемлемой стоимости resolved task ветвь останавливается; identity-only rebrand не считается результатом.

## Release artifact

- **VERIFIED FACT:** `E-01 Research Candidate` — versioned checkpoint reference, adapter pack, runtime contracts, CodeWorld schemas, policy bundle, eval dossier и provenance manifest. Это research artifact, не production deployment.
